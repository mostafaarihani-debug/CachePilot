# PRODUCT.md

## Product name
CachePilot

## Product idea
CachePilot is a modern PC cache cleaner that helps users clean cache files safely with one click.

## Target users
- Everyday PC users.
- People with low disk space.
- Users with slow or glitchy apps.
- Users who want a simple maintenance tool.
- Users who do not want technical jargon.

## User promise
"See what cache is slowing you down, understand it quickly, and clean it safely."

## Main benefits
- Frees storage.
- Helps fix glitches caused by stale cache.
- Makes cleanup easy.
- Explains each category clearly.
- Keeps scan results saved.

## Core features
- One-click scan.
- One-click cleanup.
- Cache type explanations.
- Local scan history.
- Dark modern UI.
- Safety warnings.
- Settings and exclusions.
- Summary dashboard with clear status.

## Product rules
- Never overwhelm the user with technical terms.
- Never hide risky side effects.
- Make every action understandable.
- Make cleanup feel trustworthy.
- Keep the app lightweight.

## Tone of voice
- Clear.
- Calm.
- Helpful.
- Modern.
- Confident but not aggressive.

## Value proposition
Most cleaners are either too technical or too vague. CachePilot sits in the middle: simple for beginners, but detailed enough to build trust.

## Success criteria
- User understands what was scanned.
- User understands what will be cleaned.
- Results persist after navigation.
- The UI feels like a premium PC utility.
- Cleanup is fast and reliable.
