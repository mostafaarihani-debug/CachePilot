# ui-design Skill

## Purpose
Help the agent design a polished dark desktop UI.

## Responsibilities
- Define layout and spacing.
- Choose color and typography direction.
- Create component structure.
- Improve readability.
- Keep the design modern and trustable.

## Style rules
- Use a dark dashboard aesthetic.
- Use clear hierarchy.
- Avoid white backgrounds.
- Use accent colors sparingly.
- Keep text simple.
- Make buttons obvious.

## Component patterns
- Summary cards.
- Category cards.
- Detail drawer.
- History table or timeline.
- Settings panel.
- Primary action hero section.

## Copy rules
- Use short labels.
- Avoid technical clutter.
- Make actions obvious.
- Make state changes understandable.

## Logo direction
- Minimal.
- Memorable.
- Trustworthy.
- Sweeping or shield-like.
- Works in small sizes.

## Success definition
The UI should feel like a premium optimizer app, not a generic admin tool.
