- Add category selection.
- Add confirmation for risky items.
- Log all actions.

- Add icons and gradients.
- Add better explanations.
- Add empty and loading states.

## Phase 5: History and settings
- Build scan history.
- Add exclusions.
- Add cleanup preferences.
- Add data retention controls.

## Phase 6: Quality and testing
- Validate cleanup behavior.
- Test persistence.
- Test the dark theme.
- Fix edge cases and refine copy.

## Delivery rule
Build one small working slice at a time.
