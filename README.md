# CachePilot

CachePilot is a modern desktop app that helps users scan and clean cache files in one click.

It is designed to be fast, simple, and easy to understand, with a dark modern UI and plain-language explanations for every cache type.

## What it does

- Scans common cache locations on the PC.
- Explains what each cache type is in simple language.
- Lets the user clean selected cache items or use one-click cleanup.
- Stores scan results locally so history does not disappear.
- Shows warnings for cleanup actions that may sign the user out or reset preferences.

## Why it exists

Many PC users know their computer feels slow, but they do not know what cache is safe to clean.

CachePilot solves that by making cache cleanup:
- visible,
- understandable,
- and safe.

## Main features

- One-click scan.
- One-click cleanup.
- Cache explanations for every category.
- Persistent scan history.
- Dark modern dashboard UI.
- Local-first storage.
- Safe action warnings.
- Settings and exclusions.

## Supported cache types

- Browser cache.
- Cookies and site data.
- Temporary files.
- Windows Update cache.
- DNS cache.
- Microsoft Store cache.
- Thumbnail cache.
- Icon cache.
- Prefetch cache.
- GPU and shader cache.
- App-specific cache.
- Log files.

## Design direction

CachePilot should feel like a premium PC optimizer:
- dark surfaces,
- blue and green accents,
- clear cards,
- readable text,
- smooth spacing,
- minimal clutter,
- trust-first UI.

## Safety principles

- Never remove personal files.
- Never hide risky actions.
- Always explain side effects.
- Keep cleanup choices visible.
- Prefer safe defaults.

## Development approach

Build the app in small steps:
1. App shell.
2. Scan engine.
3. Persistent storage.
4. Cleanup actions.
5. Dark UI polish.
6. History and settings.
7. Testing and refinement.

## Logo concept

Use a simple brand mark that combines:
- a cleaning sweep,
- a shield for trust,
- and a subtle system/dashboard feel.
