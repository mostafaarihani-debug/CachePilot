# SKILLS.md

## Skill system purpose
Skills are reusable instruction modules for OpenCode.

## Active skills
- cache-cleanup.
- ui-design.
- docs.

## When to use skills
### cache-cleanup
Use when scanning, classifying, explaining, or removing cache files.

### ui-design
Use when designing screens, components, themes, spacing, typography, or logo direction.

### docs
Use when writing, editing, structuring, or maintaining markdown project files.

## Skill rules
- Keep each skill focused.
- Do not overlap responsibilities too much.
- Keep instructions short and actionable.
- Use plain language.
- Include examples when helpful.

## Skill loading guidance
The agent should load only the skill needed for the current task, not all skills at once unless the task spans multiple areas.

## Priority
1. Product and safety.
2. Cache logic.
3. UX quality.
4. Documentation quality.
